1)Чтобы все заработало на компьютере должен быть 
установлен интерпретатор Python

2)Должна быть создана база данных на Postgres, 
и должен быть обеспечен доступ к ней. Для этого нужно зайти в PgAdmin,
кликнуть по базе данных, и ввести пароль от неё, если ранее он не был введен

3)В файл .env должны быть прописаны токен бота, 
имя БД, имя пользователя, пароль пользователя, 
порт на котором работает постргес (обычно 5432) и хост 
(обычно это - localhost)

4)После чего нужно запустить файл creation.py,
он выполнит создание виртуального окружение,
заполнение этого окружения библиотеками и 
апргейд базы данных до необходимого состояния

5)Для того чтобы создать пользователя, нужно запустить файл create_user.py,
после чего будет создан пользователь по имени Кирилов Сергей Владимирович, 
его лицевой счет 1211235

6)После этого, чтобы бот запустился, нужно запустить файл
run_bot.py
